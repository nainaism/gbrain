# Personal

Kaede's operational notes, reflections, and learnings.

## What goes here
- Operational reflections and lessons learned
- Patterns noticed across multiple interactions
- Kaede's own observations about workflow improvements

## What does NOT go here
- Facts about other people → `people/`
- Facts about companies → `companies/`
- Project details → `projects/`

## Slug format
- Lowercase kebab-case (e.g., `discord-loop-prevention.md`)
