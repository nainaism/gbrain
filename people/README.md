# People

This directory contains pages about real people.

## What goes here
- Team members (かえで, ハカセ, はなび, 成田さん)
- Clients, contacts, industry figures
- Anyone we interact with or need to track

## What does NOT go here
- Fictional characters → `concepts/`
- Company pages that happen to be named after people → `companies/`
- Bot personas (they go here only if they represent real people)

## Slug format
- `first-last.md` (e.g., `narita.md`, `tanaka.md`)
- Use the most common name (Japanese or English)
- One page per person, no duplicates
