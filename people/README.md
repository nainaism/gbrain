     1|# People
     2|
     3|This directory contains pages about real people.
     4|
     5|## What goes here
     6|- Team members (かえで, ハカセ, はなび, 成田さん)
     7|- Clients, contacts, industry figures
     8|- Anyone we interact with or need to track
     9|
    10|## What does NOT go here
    11|- Fictional characters → `concepts/`
    12|- Company pages that happen to be named after people → `companies/`
    13|- Bot personas (they go here only if they represent real people)
    14|
    15|## Slug format
    16|- `first-last.md` (e.g., `narita.md`, `tanaka.md`)
    17|- Use the most common name (Japanese or English)
    18|- One page per person, no duplicates
    19|
