# Concepts

Pages about technical concepts, frameworks, and patterns.

## What goes here
- AI/ML concepts (agents, RAG, embeddings, vector search)
- Software architecture patterns
- Methodologies and frameworks
- Technical terms we reference frequently

## What does NOT go here
- Specific implementations → `projects/`
- People → `people/`
- Companies → `companies/`

## Slug format
- Lowercase kebab-case (e.g., `hermes-agent.md`, `pglite.md`)
